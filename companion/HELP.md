## PTZ Live

This module connects Bitfocus Companion to PTZ Live.

Configure the PTZ Live host, port, and optional API token in the module settings.

### Actions

- Set Preview
- Set Program
- Cut
- Auto
- Enable Graphics Layer
- Enable Processor
- Select Deck Media
- Recall Preset

### Feedbacks

- Source is Preview: turns a button green when the configured source is active in preview.
- Source is Program: turns a button red when the configured source is active in program.
